reset.css
---

[![](https://img.shields.io/npm/v/@uiw/reset.css.svg)](https://www.npmjs.com/@uiw/reset.css)

A tiny modern CSS reset.

### npm

```bash
npm install @uiw/reset.css --save
```

### Usage

```js
import '@uiw/reset.css';
// or
import '@uiw/reset.css/reset.less';
```
